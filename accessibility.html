Accessibility Statement

Last updated: November 15, 2025

Our Commitment

Just For Phishing is committed to ensuring digital accessibility for people of all abilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Accessibility Features

Our platform includes the following accessibility features:

⌨️ Keyboard Navigation

All interactive elements can be accessed via keyboard

🎨 Color Contrast

Sufficient color contrast ratios for readability

📱 Responsive Design

Works on all device sizes and orientations

🔤 Readable Fonts

Clear, legible typography throughout

Current Features

  • Semantic HTML: Proper heading structure and ARIA labels where appropriate
  • Keyboard Support: Tab navigation and keyboard shortcuts
  • Focus Indicators: Clear visual focus states for interactive elements
  • Responsive Text: Text scales appropriately on different screen sizes
  • Alt Text: Descriptive alternative text for images and icons
  • Color Independence: Information not conveyed by color alone

Known Limitations

We are aware of the following accessibility limitations and are working to address them:

  • Some interactive simulations may not be fully screen reader compatible
  • Animated elements may affect users with motion sensitivity
  • Some quiz interactions require precise mouse/touch control

Conformance Status

We aim to conform to WCAG 2.1 Level AA standards. As an open-source project under active development, we continuously work to improve accessibility compliance.

Feedback & Assistance

We welcome feedback on the accessibility of Just For Phishing. If you encounter accessibility barriers:

  • Visit our Contact page to report issues
  • Open an issue on GitHub with the "accessibility" label
  • Describe the specific issue and the steps to reproduce it
  • Include information about your assistive technology (if applicable)

Compatibility

Just For Phishing is designed to be compatible with:

  • Modern web browsers (Chrome, Firefox, Safari, Edge)
  • Screen readers (NVDA, JAWS, VoiceOver)
  • Keyboard-only navigation
  • Browser zoom up to 200%
  • Mobile devices with touch interfaces

Third-Party Content

Some pages may link to external resources. We are not responsible for the accessibility of third-party websites.

Continuous Improvement

This is an open-source project, and we encourage contributions that improve accessibility. If you're a developer interested in helping make Just For Phishing more accessible, please visit our GitHub repository.

Contact Us

For accessibility questions or to report issues, please contact us. We typically respond within 3-5 business days.
